# Device

An <Term name="Device"/> is a physical or virtual entity that supports 
interaction between other parts of the LCPF and the outside world.  

It usually represents some hardware component either included on
 or connected to the <Term name="Controller"/>.  A <Term name="Device"/> can be 
"connected" as part of a <Term name="Shield"/> or as an individual component 
over a standardized intertconnect system like using with such as a sensor, actuator, or any other component that interacts with the system.
