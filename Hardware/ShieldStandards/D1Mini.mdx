# D1 Mini Shield Standard

The D1 Mini <Term name="ShieldStandard"/>  based on the pin layout from the [original Wemos ESP8266 D1 Mini](<https://www.wemos.cc/en/latest/d1/d1_mini.html>)

![D1 Mini Pin Layout](https://www.wemos.cc/en/latest/_static/boards/d1_mini_v4.0.0_5_16x9.png)

There are numerous newer boards which support the same pinout (usually extended with additional pins) baseed on the ESP32 family of chips, which are supported by LCPF.   These include the [Lolin S2 Mini](<https://www.wemos.cc/en/latest/s2/s2_mini.html>) and [Lolin S3 Mini](<https://www.wemos.cc/en/latest/s3/s3_mini.html>).
