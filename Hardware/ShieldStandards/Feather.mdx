# Feather Shield Standard

> [!WARNING]
>
> This page is informational - the LCPF does NOT (yet) directly support
> any Feather compatible <Term name="Shield"/> or <Term name="Controller"/> boards.

The [Feather shield standard](https://learn.adafruit.com/adafruit-feather/feather-specification) is a <Term name="ShieldStandard"/> created by [Adafruit](https://www.adafruit.com/).  It has one 12 pin and one 16 pin header for a total of 28 pins.

| Pins | Usage | Notes | | Pin Names |
| --- | --- | --- | --- | --- |
| 4 | Analog IO |  |  | A0, A1, A2, A3 |
| 2 | Analog _or_ GPIO |  |  | A4, A5 / D24, D25 |
| 7 | GPIO |  |  | D5, D6, D9, D10, D11, D12, D13 |
| 2 | UART _or_ GPIO |  |  | RX, TX / D0, D1 |
| 3 | SPI |  |  | MI, MO, SCK |
| 2 | I2C |  |  | SDA, SCL |
| 2 | Power |  |  | 3.3F,  GND |
| 2 | Control |  |  | En, RST, |
| 3 | Other |  |  | ARef, Bat,  Free |
