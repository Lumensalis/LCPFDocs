# Xiao Shield Standard

> [!WARNING]
>
> This page is informational - the LCPF does NOT (yet) directly support
> any Xiao compatible <Term name="Shield"/> or <Term name="Controller"/> boards.


The [Xiao shield standard](https://www.seeedstudio.com/xiao-series-page) was 
created by [seeed studio](https://www.seeedstudio.com).  It has two 7 pin 
headers for a total of 14 pins.

![Xiao Pinout](https://files.seeedstudio.com/wiki/Seeeduino-XIAO/img/Seeeduino-XIAO-pinout-1.jpg)


[QT Py ESP32C3](https://www.adafruit.com/product/5376) boards.

| Pins | Usage | Notes | | Pin Names |
| --- | --- | --- | --- | --- |
| 1 | IO _or_ DAC |  |  | DAC, A0, D0  |
| 3 | IO |  |  | A1, A2, A3 / D1, D2, D3 |
| 2 | I2C _or_ IO   |  |  | SDA, SCL / A4, A5 / D4, D5 |
| 2 | UART _or_ IO  |  |  | RX, TX / D6, D7 / A6, A7 |
| 3 | SPI _or_ IO   |  |  | MOSI, MISO, SCK / A8, A9, A10 / D8, D9, D10 |
| 3 | Power |  |  | 3V3, 5V, GND |
