# NeoPixel / Ws2812 LEDS

[Connections](connections.mdx)

## Module Types

NeoPixels come in a wide variety of packageshttps://www.amazon.com/Tapes-Adhesives-Sealants/b/?ie=UTF8&node=256225011&ref_=sv_indust_8

### Single
  - Through Hole - 4 leg LED package
  - .mdx - surface mount package typically added to a PCB
  - PCB - an .mdx NeoPixel mounted on a small PCB to make it easier to connect

### Multiple

  - Stick
  - Strip
  - Strand
  - Ring
  - Matrix
